In my prevoius project it was all about CSS and HTML and what we learn from it. But this project takes place in unit 2 where we learn about git and github, real quick git is like a software control that lives in your laptop while github is the website, We made the project all in git. My teacher, Mr.Muller, assigned us partners to work with in git to practice our git pull and push, but also making a good story where every choice lead to a outcome. Me and my partner would use the IDE to git pull and git push our edits into the main cloud and then pull the edits they made.
